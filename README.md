# 📝 Daily Task Journal

A simple web app built with **HTML, CSS, and JavaScript** to manage daily tasks.  
It lets you add, complete, delete tasks, and keeps a history of all actions.

---

## ✨ Features
- Add new tasks with timestamps
- Mark tasks as completed (with automatic history logging)
- Delete tasks (also logged in history)
- Task history section showing all actions (added, completed, deleted)
- Stylish input box and buttons with hover effects
- LocalStorage support (tasks and history persist after refresh)

---

## 🚀 How to Use
1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Start adding tasks!
4. Completed and deleted tasks will appear in the **History** section.
